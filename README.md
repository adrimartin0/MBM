# Web de muestra para Codex

Plantilla simple para practicar con Codex, GitHub y GitHub Pages.

## Archivos

- `index.html`: estructura principal.
- `styles.css`: estilos visuales.
- `script.js`: menú responsive.
- `CODEX_INSTRUCCIONES.md`: prompt preparado para pedir mejoras a Codex.

## Cómo publicarla con GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. Entra en `Settings`.
3. Entra en `Pages`.
4. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda.
6. Espera a que GitHub genere la URL pública.
